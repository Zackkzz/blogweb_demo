# Personal Website

A personal website built with Next.js, featuring Home, About, Projects, and Blog pages. Content can be edited via the Admin panel.

## Getting Started

1. Install dependencies: `npm install`

2. Run the development server: `npm run dev`

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

4. Go to /admin to edit content.

## Features

- Home page
- About page
- Projects page
- Blog page
- Admin panel for editing content

All content is stored in `data/content.json` and served via API routes.
