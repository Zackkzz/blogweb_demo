# Public Assets

## Profile Image

To add your profile image:

1. Place your profile image in this directory (`frontend/public/`)
2. Name it `zack-profile.jpg` (or update the filename in `frontend/app/page.tsx`)
3. Supported formats: JPG, PNG, WebP
4. Recommended size: 150x150 pixels or larger (will be automatically resized)

The image will be displayed in the bottom right corner of the home page.

If the image is not found, a default placeholder will be shown.
